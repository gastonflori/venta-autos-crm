# Sote CRM - Venta de Autos

Proyecto nuevo e independiente para administrar una agencia de venta de autos.

## Incluye

- Login con sesiones.
- Panel administrativo estilo Sote CRM.
- Stock de vehiculos.
- Clientes y leads.
- Pipeline de ventas.
- Gestoria.
- Finanzas.
- WhatsApp / plantillas.
- Configuracion.
- Persistencia SQLite.
- Backup JSON.
- Deploy preparado para Railway.

## Correr local

```bash
npm start
```

URL:

```text
http://localhost:4174
```

Usuario inicial:

```text
admin@autos.app
admin1234
```

En produccion configurar `DATA_DIR=/data` y variables `BOOTSTRAP_ADMIN_*`.
